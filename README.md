# Books

Book tracker powered by [Bookshelf Action](https://github.com/bhaumikmistry/bookshelf-action).

**API:** https://bhaumikmistry.github.io/books/api.json

## How to use

- **Add a book**: Open an issue with the book title
- **Track progress**: Comment `50%` or `150/300` (page/total)
- **Finish**: Close the issue
- **Rate**: Add a label like `rating: 4/5`
